# References

Canonical reference library for thesis writing and evidence audits.

## Organization Standard
- Keep canonical source files at the `references/` root for path stability.
- Keep transcript assets under `references/onocoy_transcripts/`.
- Use `citation_manifest.csv` as the primary citation inventory.
- Use canonical snake-case filenames for all new additions.
- Do not add alternate filename variants for the same document.
