# Graph Route Usage Log Template

Use this template to capture practical feedback from real thesis sessions and keep route decisions evidence-based.

## Session preflight checklist

- [ ] `python3 scripts/knowledge_skill_graph.py --validate`
- [ ] `python3 scripts/knowledge_skill_graph.py --validate-policy`
- [ ] `python3 scripts/knowledge_skill_graph.py --task <task-id>`

## Route usage log entry (copy/paste)

```markdown
### Session Log
- Date:
- Task ID:
- Context (what I was editing):
- Route usefulness (1-5):
- Noise/friction (1-5):
- What felt off semantically?:
- Time saved (estimate):
- Follow-up change idea (optional):
```
